# Figures

This folder contains publication-quality figures generated during the LUAD transcriptomic analysis.

## Figures Included

- Volcano Plot
- Gene Expression Heatmap
- Biomarker Expression Bar Chart
- Gene Category Pie Chart
- Expression Histogram
- Expression vs. Log₂ Fold Change Scatter Plot

These figures summarize the major transcriptomic patterns and candidate biomarkers identified during the computational analysis.
